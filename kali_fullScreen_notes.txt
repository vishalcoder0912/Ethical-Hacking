2. How to make Kali Linux full screen
This depends on whether you are using VirtualBox or VMware.

If using VirtualBox
Press:

Right Ctrl + F
or use the menu:

View → Full-screen Mode
If it does not resize properly, install guest additions:

sudo apt update
sudo apt install -y build-essential dkms linux-headers-$(uname -r)
Then in VirtualBox menu:

Devices → Insert Guest Additions CD Image
Run:

sudo mkdir /media/cdrom
sudo mount /dev/cdrom /media/cdrom
sudo sh /media/cdrom/VBoxLinuxAdditions.run
reboot
If using VMware
Press:

Ctrl + Alt + Enter
Or menu:

View → Full Screen
If resolution is wrong, install tools:

sudo apt install open-vm-tools-desktop
reboot
